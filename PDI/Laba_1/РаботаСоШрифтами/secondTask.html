Кодирование информации

Чтобы информация могла обрабатываться компьютером, она должна быть сначала закодирована

Кодирование - представление символов одного алфавита при помощи символов другого алфавита

Для кодирования используется двоичная система счисления, в алфавите которой только два символа - 0 или 1. Символ 1 соответтствует наличию сигнала, символ 0 - его отсутствию. 

Один двоичный символ получил нгазвание бит, от английской аббревиатуры binary digit - бинарная (двоичная) единица.
Любой
		символ,
			букву,
				цифру
можно представить в виде
			последовательности битов.

Для определния количества инофрмации введены единицы иземрения:

		1 Бит = 1 разряд(может принимать значения 0 или 1)
				1 Байт = 8 бит
			 1 кБайт = 1024 Байт = 210 Байт
			 1 мБайт = 1024 кБайт = 220 Байт
			 1 ГБайт = 1024 МБайт = 230 Байт
			 1 ТБайт = 1024 ГБайт = 240 Байт
